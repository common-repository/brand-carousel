Responsive Brand Carousel/Image Carousel. Easy to use.

== Description ==

Responsive Brand Carousel/Image Carousel is easy to use. You can add as many slide as you want. 
If you want to modify or want to add extra features, please contact the owner faferdaus@gmail.com.

= Uses =
Use this shortcode to show the carousel [brand-carousel]

= Privacy notices =

With the default configuration, this plugin, in itself, does not:

* track users by stealth;
* send any data to external servers;
* use cookies.

== Installation ==

1. Upload the entire `Brand Carousel` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Brands Carousel** menu in your WordPress admin screen.
